## Amazon E-Commerce C++
- A C++ console-based e-commerce application that simulates an online store with products, orders, and shipping options. Users can browse products, add them to orders, select shipping methods, and view total costs in a menu-driven interface.

## Features
- Load products from a data file (file.dat)
- Supports multiple product types: Clothing, Tools, and Food
- Create and manage orders with quantities
- Choose between standard and express shipping
- Menu-driven console interface with input validation

## Technologies
- C++
- File I/O for product data
- Object-oriented programming with classes and inheritance
